## ğŸš€ Quick Start

### Prerequisites
- Node.js 18 or higher
- npm or yarn package manager

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/nazmul-haque-nihal/Programming-Tools.git
cd Programming-Tools
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

4. **Open your browser**
Visit [http://localhost:5173](http://localhost:5173) to see the application

### Production Build

```bash
# Build for production
npm run build

# Preview production build locally
npm run preview
```

## ğŸ¨ Customization

### Adding New Tools

To add a new tool to the toolkit:

1. Create a new component in the `src/tools/` directory
2. Add the tool configuration to `src/data/toolCategories.ts`
3. Import and register the component in the appropriate category

### Theme Customization

The application uses Tailwind CSS for styling. You can customize:
- Colors in `tailwind.config.js`
- Global styles in `src/index.css`
- Component-specific styles in individual components
